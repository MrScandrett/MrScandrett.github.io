Ideas & Inspiration

Recipe Book

Pick a recipe, gather your parts, and build something real. Recipes span beginner to advanced across electronics, circuits, coding, and physical making.

Community schematic source: Arduino Project Hub is a strong place to see what people are building together, study shared schematics, and gather ideas from a wider maker community.

🔌 Electronics & Arduino

Hands-on builds using an Arduino Uno, breadboards, and basic components.

💡

Blink

Beginner

Make an LED flash on and off — the "Hello, World!" of electronics. Write your first Arduino sketch and learn how to control a digital output pin with digitalWrite() and delay().

  • Arduino Uno
  • LED
  • 220Ω resistor
  • Breadboard
  • USB cable
🚦

Traffic Light

Beginner

Build a working traffic light sequence with three LEDs. Use delay() and a looping sketch to cycle through red, yellow, and green on a realistic timer.

  • Arduino Uno
  • Red LED
  • Yellow LED
  • Green LED
  • 3 × 220Ω resistors
  • Breadboard
  • Jumper wires
🔘

Push-Button LED

Beginner

Wire a push button so pressing it turns on an LED. Introduces digital input, pull-down resistors, and reading a pin state with digitalRead().

  • Arduino Uno
  • LED
  • Push button
  • 10kΩ resistor
  • 220Ω resistor
  • Breadboard
  • Jumper wires
🎵

Buzzer Melody

Intermediate

Play a recognizable tune through a passive buzzer using the tone() function. Map musical notes to their frequencies, set durations, and compose a short melody in code.

  • Arduino Uno
  • Passive buzzer
  • Jumper wires
  • USB cable
🌡️

Temperature Reader

Intermediate

Read the room temperature with a DHT11 sensor and print it to the Serial Monitor. Covers analog sensors, installing a library, and printing formatted data over USB.

  • Arduino Uno
  • DHT11 sensor
  • 10kΩ resistor
  • Breadboard
  • Jumper wires
  • USB cable
🦾

Servo Sweep

Intermediate

Make a servo motor arm sweep back and forth smoothly. Learn about PWM signals, the built-in Servo library, and how code can create real physical movement.

  • Arduino Uno
  • SG90 servo motor
  • Jumper wires
  • USB cable

⚡ Snap Circuits

Quick builds using the Snap Circuits kit — no soldering, no loose wires. Snap parts together and see it work immediately.

💨

Spinning Fan

Beginner

Connect the battery holder, switch, and motor fan to build a working electric fan. One of the simplest complete circuit builds in the kit — perfect for a first day.

  • Battery holder (B1)
  • On/off switch (S1)
  • Motor + fan blade (M1)
  • Base grid
  • Snap wires
🔔

Alarm Bell

Beginner

Wire a spring switch to a buzzer to create a simple alarm. When the spring is pressed, the circuit closes and the alarm sounds — a great intro to switches as sensors.

  • Battery holder (B1)
  • Buzzer (SP1)
  • Spring switch (S3)
  • Snap wires
🌅

Light-Activated Switch

Intermediate

Use a photoresistor to turn on an LED when it gets dark. A real-world model of how night lights and streetlamps work. Introduces transistors as amplifiers and switches.

  • Battery holder (B1)
  • Photoresistor (R6)
  • LED lamp (L1)
  • Transistor (Q1)
  • Resistor snaps
  • Snap wires
📻

AM Radio Receiver

Advanced

Build a working AM radio that picks up real broadcasts from your area. One of the most complex builds in the kit — a fantastic capstone challenge for advanced students.

  • Battery holder (B1)
  • AM radio IC (U1)
  • Speaker (SP1)
  • Variable capacitor (VC)
  • Antenna coil
  • Snap wires

Dice Roller

Beginner

Roll a virtual die that shows a random number from 1 to 6 on each click. Practice Math.random(), Math.floor(), and displaying a result on screen.

  • HTML file
  • JavaScript
  • Text editor
  • Browser

Color Switcher

Beginner

Create color buttons that change the entire page background when clicked. Great practice for reading style properties and wiring buttons to JavaScript actions.

  • HTML file

  • Text editor

Name Tag Generator

Beginner

Type your name into a form field and instantly generate a styled name badge. Introduces HTML form inputs, reading .value, and applying dynamic CSS.

  • HTML file

  • Text editor

Pixel Painter

Intermediate

Build a grid of squares that fill with a chosen color when clicked — a simple pixel-art editor from scratch. Covers CSS Grid, generating elements with JS loops, and event delegation.

  • JavaScript
  • Text editor

Snake Game

Advanced

Build the classic Snake from scratch on an HTML canvas. Covers game loops with requestAnimationFrame, keyboard input, collision detection, and growing arrays. A strong capstone project.

🔧 Physical Making

Engineering design challenges using everyday materials. Think, build, test, and improve your design.

🌉

Straw Bridge

Beginner

Using only straws and masking tape, build the strongest bridge that can span a 30 cm gap. Test it by stacking pennies in the center until it fails — then redesign.

  • 20 plastic straws
  • Masking tape
  • Ruler
  • Pennies (test weight)
🗼

Paper Tower

Beginner

Using one sheet of paper and scissors only — no tape, no glue — build the tallest freestanding tower that can hold a marshmallow on top for 10 full seconds.

  • 1 sheet of paper
  • Scissors
  • 1 marshmallow
  • Ruler
🚗

Rubber Band Car

Intermediate

Build a car chassis from cardboard and skewers, then power it with a wound rubber band. Explore stored elastic energy, friction, and wheel alignment to get the furthest distance.

  • Cardboard strip
  • 4 bottle caps
  • 2 bamboo skewers
  • Rubber band
  • Tape
🪂

Egg Drop Pod

Advanced

Design a pod that protects a raw egg dropped from a second-floor window. Apply Newton's laws — reduce the impact force by increasing stopping time and distance. No cracks wins.

  • Raw egg
  • Plastic bags
  • Bubble wrap
  • Straws
  • Tape & string
