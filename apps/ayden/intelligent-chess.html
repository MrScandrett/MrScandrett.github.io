Intelligent Chess Game

Play chess in your terminal against a powerful minimax AI with alpha-beta pruning.

Setup & Play

Run the following commands in your terminal:

pip install -r chess_requirements.txt
python3 chess_game.py

How to Play

  • You are White (uppercase pieces like ♔ ♕ ♖ ♗ ♘ ♙)
  • AI is Black (lowercase pieces)
  • Enter moves in UCI format: e2e4, g1f3, etc.
  • Type quit to exit at any time

Features

  • ✓ Minimax algorithm with alpha-beta pruning
  • ✓ Material-based board evaluation
  • ✓ Adjustable AI depth (default: 2)
  • ✓ Full legal move validation
  • ✓ Check, checkmate, and stalemate detection

Files

  • chess_game.py — Main game script
  • chess_requirements.txt — Python dependencies
