# Horse Field Demo

Playable third-person horse demo using:
- Horse asset: `horse-riggedgame-ready/source/horse.glb`
- Scene assets: `low-poly-winter-tree-pack` (OBJ + winter textures)

## Run with Live Server

1. Open this folder in VS Code.
2. Install the **Live Server** extension (if needed).
3. Right-click `index.html` and choose **Open with Live Server**.

## Controls

- `W`: Move forward
- `S`: Reverse
- `A` / `D`: Turn
- Mouse drag: Orbit camera
- Mouse wheel: Zoom
