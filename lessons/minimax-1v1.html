AI Origins · Lesson 05

1v1 Game AI: Minimax

Build adversarial search for a two-player game and show why minimax guarantees optimal play in small states.

Learning Goals

  • Goal 1Explain maximizing and minimizing turns
  • Goal 2Represent game state and legal moves
  • Goal 3Score terminal states for win loss draw
  • Goal 4Measure search depth and branching factor

Build Tasks

  • Task 1Implement minimax for Tic-Tac-Toe
  • Task 2Return both score and best move
  • Task 3Add UI controls for human vs AI
  • Task 4Log move decisions for debugging

Lesson Outputs

Each team should publish a short reflection and demo link at the end of class.
