Limit Notes

WHAT THIS IS

A perceptron is a single-layer linear classifier. With two inputs, it draws one straight dividing line in 2D feature space.

WHAT IT CAN'T DO

It cannot solve XOR with two inputs, because XOR is not linearly separable. No single line can split XOR classes correctly.

HOW TO USE IT LIKE IT'S 1958

  • SET CHANNELS TO (1,0). RUN.
  • ADJUST W1 UPWARD UNTIL OUTPUT FLIPS.
  • USE SMALL LEARNING RATES.
  • TRAIN IN SHORT BURSTS; READ THE LOGBOOK.
  • SWITCH DATASET TO XOR. OBSERVE FAILURE MODE.
