AI Origins · Lesson 06

Chess AI Core

Apply minimax with alpha-beta pruning to a simplified chess environment.

Learning Goals

  • Goal 1Explain pruning and why it speeds search
  • Goal 2Build a lightweight board evaluation function
  • Goal 3Compare shallow and deeper search behavior
  • Goal 4Identify heuristic tradeoffs in move quality

Build Tasks

  • Task 1Use a reduced board or piece set for class scope
  • Task 2Implement legal move generation
  • Task 3Add alpha-beta pruning to minimax
  • Task 4Tune evaluation weights for material and position

Lesson Outputs

Each team should publish a short reflection and demo link at the end of class.
