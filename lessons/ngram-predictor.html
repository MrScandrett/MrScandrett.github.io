AI Origins · Lesson 08

N-gram Text Predictor

Recreate a pre-LLM language modeling baseline using n-gram probability tables.

Learning Goals

  • Goal 1Tokenize text into clean sequences
  • Goal 2Build unigram bigram and trigram counts
  • Goal 3Predict next tokens from context windows
  • Goal 4Evaluate output quality with simple metrics

Build Tasks

  • Task 1Ingest a small classroom corpus
  • Task 2Compute frequency and conditional probability tables
  • Task 3Generate text from a seed phrase
  • Task 4Compare n values and smoothing choices

Lesson Outputs

Each team should publish a short reflection and demo link at the end of class.
