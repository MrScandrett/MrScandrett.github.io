AI Origins · Lesson 07

A* Pathfinding Agent

Implement A* search to navigate grid maps with blocked cells and weighted movement.

Learning Goals

  • Goal 1Differentiate g h and f costs
  • Goal 2Choose and justify a heuristic
  • Goal 3Trace the open and closed sets
  • Goal 4Compare A* to uninformed search

Build Tasks

  • Task 1Create a grid map with start and goal nodes
  • Task 2Implement priority queue expansion by f score
  • Task 3Reconstruct final path from parent pointers
  • Task 4Visualize visited nodes and final route

Lesson Outputs

Each team should publish a short reflection and demo link at the end of class.
