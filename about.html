About This Hub

This site is a static, GitHub Pages-ready project library for showcasing student work with searchable discovery and rich detail pages.

Design Approach

The Home page uses streaming-style curated rows, Browse supports fast faceted filtering, and each project opens a dedicated detail page with media and metadata.

All content is sourced from one file: data/projects.json. Adding or editing projects there updates all pages.

Accessibility + Performance

  • Keyboard-focusable cards and controls.
  • Visible focus states and ARIA labels for scroll rows.
  • Lazy-loaded images and shared in-memory data caching.
  • Filter state preserved in URL query params.

Recommended Publishing Safety

  • Use aliases instead of full names.
  • Avoid personal data in descriptions or media.
  • Review outbound links before publishing.

Static architecture: no backend, no login, no runtime dependencies.
